# 任务 2: 用户系统开发

## 描述
实现用户注册登录功能,包括微信授权登录、手机验证码登录、游客模式

## 详细信息
需要实现:
1. 微信授权登录(APP和小程序)
2. 手机号+验证码登录
3. 游客模式(先体验后注册)
4. 游客模式限制(历史记录不保存、仅可卜卦2次、不可用付费功能)
5. 游客卜卦2次后引导注册
6. 个人中心页面

## 验收标准
[
  "微信授权登录功能正常",
  "手机验证码登录功能正常",
  "游客模式可以正常体验",
  "游客限制正确实施",
  "引导注册弹窗正确触发",
  "个人中心页面功能完整"
]

## 测试策略
测试各种登录方式的完整流程,验证游客模式限制和引导注册逻辑
