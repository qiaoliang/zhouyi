## 1. 项目概述

### 1.1 产品定位

周易通是一款面向周易零基础新手及易学爱好者的移动应用，提供周易知识科普、零基础入门课程和专业的金钱课卜卦解卦工具。

### 1.2 目标用户

| 用户类型 | 特征描述 |
|----------|----------|
| 主要用户 | 纯周易零基础的新手，对周易文化感兴趣但缺乏系统学习渠道 |
| 次要用户 | 有一定基础的易学爱好者，需要专业的卜卦工具 |

### 1.3 核心价值

- 降低周易学习门槛，提供系统化的入门路径
- 提供专业可靠的金钱课卜卦工具
- 传承和推广周易传统文化

## 3. 功能需求

### 3.1 功能模块总览

```
周易通 APP
├── 用户系统
│   ├── 注册登录（微信授权、手机验证码）
│   ├── 游客模式
│   └── 个人中心
├── 卜卦功能
│   ├── 金钱课起卦（摇铜钱动画）
│   ├── 一键起卦
│   ├── 基础解卦（免费）
│   ├── 详细解卦（付费）
│   └── 精准解卦（付费）
├── 学习路径
│   ├── 六十四卦解读方法
│   ├── 如何起卦
│   ├── 八卦介绍
│   ├── 如何看卦
│   └── 周易起源与历史
├── 每日一卦
│   ├── 当日卦象推送
│   └── 历史记录（会员专享）
├── 历史记录
│   ├── 卜卦记录查看
│   └── 记录管理（付费功能）
└── 辅助功能
    ├── 分享功能（后续版本）
    ├── 卦象查询（后续版本）
    └── 大师咨询（后续版本）
```

### 3.2 用户系统

#### 3.2.1 注册登录

| 功能项 | 说明 |
|--------|------|
| 微信授权登录 | 支持微信一键授权登录（APP和小程序） |
| 手机号登录 | 支持手机号+验证码登录 |
| 游客模式 | 支持游客先体验后注册 |

#### 3.2.2 游客模式限制

| 限制项 | 说明 |
|--------|------|
| 历史记录 | 不保存 |
| 学习进度 | 不保存 |
| 卜卦次数 | 仅可体验2次 |
| 付费功能 | 不可使用 |

#### 3.2.3 引导注册

| 触发条件 | 引导方式 |
|----------|----------|
| 游客卜卦2次后 | 弹窗提示 + 首页提示条 |
| 使用付费功能时 | 弹窗提示功能为会员专享 |

#### 3.2.4 个人中心（我的页面）

| 功能模块 | 说明 |
|----------|------|
| 会员信息/购买入口 | 显示会员状态、到期时间、购买入口 |
| 学习进度展示 | 显示学习路径完成进度 |
| 设置 | 推送开关、深色模式切换等 |
| 联系客服 | 客服咨询入口 |
| 意见反馈 | 用户意见反馈入口 |
| 注销账号 | 支持用户注销并删除所有数据 |

### 3.3 卜卦功能

#### 3.3.1 金钱课起卦

| 起卦方式 | 说明 |
|----------|------|
| 摇铜钱动画 | 模拟真实摇铜钱，用户点击按钮或摇晃手机，连续摇6次生成卦象 |
| 一键起卦 | 简化流程，一次性生成卦象 |

两种方式均支持，用户可自行选择。

#### 3.3.2 基础解卦（免费）

| 内容项 | 说明 |
|--------|------|
| 卦象名称 | 如"乾为天" |
| 卦辞原文 | 传统通行本（王弼注本） |
| 爻辞原文 | 包含六爻爻辞 |
| 白话解释 | 卦辞和爻辞的白话解释 |

#### 3.3.3 详细解卦（付费）

| 内容项 | 说明 |
|--------|------|
| 包含基础解卦所有内容 | 卦象名称、卦辞、爻辞原文及白话解释 |
| 变卦分析 | 分析爻变后的卦象变化 |
| 互卦分析 | 分析互卦的含义 |
| 应期分析 | 预测应验时间 |
| 其他高级分析 | 专业解卦所需的其他要素 |

#### 3.3.4 精准解卦（付费）

需要收集以下用户信息：

| 信息项 | 说明 |
|--------|------|
| 姓名 | 用户姓名 |
| 性别 | 男/女 |
| 出生日期 | 用于八字分析 |
| 占问事项 | 如事业、感情、健康等 |

#### 3.3.5 付费功能引导

非会员使用详细解卦时，弹窗提示"此功能为会员专享"，包含以下选项：
- 查看会员权益
- 立即开通
- 暂不开通

### 3.4 学习路径

#### 3.4.1 课程模块（按顺序）

| 序号 | 课程名称 | 内容量 |
|------|----------|--------|
| 1 | 六十四卦解读方法 | 1-3分钟阅读 |
| 2 | 如何起卦 | 1-3分钟阅读 |
| 3 | 八卦介绍 | 1-3分钟阅读 |
| 4 | 如何看卦 | 1-3分钟阅读 |
| 5 | 周易起源与历史 | 1-3分钟阅读 |

#### 3.4.2 内容形式

| 形式 | 说明 |
|------|------|
| 文字 | 课程主要内容 |
| 配图 | 辅助理解的图片 |
| 示意图解 | 八卦、六十四卦等图示 |
| 互动练习 | 每课后的选择题测验 |
| 小测验 | 不强制答对，辅助学习检验 |

#### 3.4.3 学习完成奖励

完成全部学习路径后，解锁特殊功能：
- 免费查看1次详细解卦

### 3.5 每日一卦

#### 3.5.1 推送设置

| 设置项 | 说明 |
|--------|------|
| 推送时间 | 每天早上8点 |
| 推送内容 | 当日卦象 + 卦辞 |

#### 3.5.2 页面功能

| 功能项 | 说明 |
|--------|------|
| 当日卦象 | 显示当天推送的卦象和解释 |
| 历史记录 | 付费会员可翻看历史每日一卦 |
| 数据保存 | 不保存，过期失效 |

#### 3.5.3 推送交互

用户点击推送通知后，直接进入当天卦象的解释页面。

### 3.6 历史记录

#### 3.6.1 记录展示

| 展示形式 | 卡片式 |
|----------|--------|
| 卡片内容 | 日期 + 卦象名称 + 卦象符号 + 简短结果 |

#### 3.6.2 用户权限差异

| 用户类型 | 保存数量 | 保存时间 | 删除功能 |
|----------|----------|----------|----------|
| 免费用户 | 仅1条 | 1个月 | 不可删除 |
| 付费用户 | 无限制 | 永久 | 单条删除或清空全部 |

#### 3.6.3 数据同步

- 支持云端同步
- 多设备登录后数据自动同步
- 支持离线查看历史记录

### 3.7 付费会员

#### 3.7.1 会员权益

| 权益项 | 说明 |
|--------|------|
| 详细解卦 | 变卦、互卦、应期等高级分析 |
| 精准解卦 | 基于个人信息的精准解读 |
| 收藏功能 | 收藏特定的卦象解释 |
| 解除次数限制 | 无卜卦次数限制 |
| 无广告 | 免费版有广告，会员无广告 |
| 专属客服 | 专属客服服务 |
| 每日一卦历史 | 可查看历史每日一卦记录 |
| 历史记录管理 | 无限保存历史记录，支持删除 |

#### 3.7.2 定价策略

| 付费方式 | 价格 |
|----------|------|
| 按次详细解卦 | 10元/次 |
| 会员月费 | 30元/月 |
| 会员年费 | 300元/年（节省17%） |
| 大师咨询 | 100元/次（后续版本） |

#### 3.7.3 会员购买页面

| 展示内容 | 说明 |
|----------|------|
| 会员权益列表 | 详细展示所有会员权益 |
| 价格对比 | 月费30元、年费300元、节省百分比 |
| 用户评价 | 好评展示（可选） |

### 3.8 辅助功能（后续版本）

#### 3.8.1 分享功能

| 支持平台 | 微信好友、朋友圈 |
|----------|------------------|
| 分享内容 | 卜卦结果 |

#### 3.8.2 卦象查询

支持以下搜索方式：
- 按卦名搜索（如"乾卦"）
- 按卦象符号搜索（如"䷀"）
- 按关键词搜索卦辞/爻辞内容

#### 3.8.3 易学大师在线咨询

| 咨询方式 | 预约制一对一 |
|----------|--------------|
| 收费方式 | 按次收费，100元/次，不限时长 |

## 4. 非功能需求

```

    ├── 卜卦（首页）
    ├── 学习
    ├── 每日一卦
    ├── 历史
    └── 我的
```

#### 4.1.3 启动页

- 显示APP名称"周易通"
- 传统中国风元素
- 每日一句易经名言

- 支持浅色/深色模式切换
- 用户可手动切换

### 4.3 性能要求

| 指标 | 要求 |
|------|------|
| 响应时间 | 页面加载时间 < 2秒 |
| 并发用户 | 支持1000+并发用户 |
| 数据同步 | 多设备数据同步延迟 < 3秒 |

### 4.4 安全要求

| 安全项 | 说明 |
|--------|------|
| 用户数据 | 加密存储 |
| 通信安全 | HTTPS 加密传输 |
| 隐私保护 | 符合个人信息保护法要求 |
| 用户注销 | 支持账号注销及数据删除 |

### 4.5 离线功能

| 功能 | 离线支持 |
|------|----------|
| 已学习内容查看 | 支持 |
| 卜卦历史查看 | 支持 |
| 起卦 | 支持（离线起卦） |
| 解卦 | 需联网（服务器处理） |

### 4.6 数据同步

- 支持云端同步
- 多设备登录后数据自动同步
- 同步内容：卜卦历史、学习进度、收藏等
