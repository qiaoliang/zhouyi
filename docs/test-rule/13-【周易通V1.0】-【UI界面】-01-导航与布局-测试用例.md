## 底部导航栏

### 导航栏切换

#### 导航栏-点击切换Tab ZYT-UI-001

**优先级**：高
**测试类型**：UI

**前置条件**：
1. APP已启动
2. 在首页（卜卦页面）

**测试步骤**：
1. 查看底部导航栏
2. 依次点击5个Tab：
   - 卜卦（首页）
   - 学习
   - 每日一卦
   - 历史
   - 我的

**预期结果**：
1. 底部导航栏固定显示
2. 每个Tab有图标和文字
3. 点击后图标高亮（激活状态）
4. 页面切换流畅
5. 切换动画自然
6. 无卡顿或闪退

**备注**：
- 5个Tab导航
- 中国风设计
- 关联需求：PRD-4.1.2
